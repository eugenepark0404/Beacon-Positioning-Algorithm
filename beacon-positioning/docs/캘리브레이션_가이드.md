# 캘리브레이션 가이드 (트랙3 → config 주입 절차)

목적: 문헌 placeholder 값을 실측값으로 교체한다. **코드 수정 없이 JSON만 바꾼다.**

## 1. 교체 대상 (유도서 §7과 동일)

| 항목 | config 키 | 현재 placeholder | 실측 방법 |
|------|-----------|------------------|-----------|
| 경로손실지수 n | `path_loss.n` | 2.0 | §2 회귀 |
| A = RSSI@1m | `path_loss.a_dbm` | -59.0 | §2 회귀 (절편) |
| 차폐 감쇠량 | `shadow.shadow_atten_db` | 7.0 dB | §3 차폐 실험 |
| 차폐 급락 임계 | `shadow.shadow_drop_db` | 4.5 dB | §3 (감쇠량의 ~2/3 권장) |
| 칼만 R | `filter.kalman_r` | 4.0 | §2에서 1m 정지 측정 분산 |
| PF sigma_ble | `pf.sigma_ble_m` | 2.0 m | 시뮬/실측 측위 오차 sigma |

## 2. 경로손실 계수 (n, A) 실측 — # ref: P1 §2, P3 §III-C, 유도서 §1.4

1. 비콘 1개를 1.0~1.5m 높이에 고정 (수신 폰과 같은 높이 — P3 §IV-A.1).
2. LoS 확보 상태로 거리 0.5, 1, 2, 3, 4, 6, 8, 12 m 각각에서 **100샘플** 수집
   (P3 관례). 각 거리에서 폰 방향 일정하게.
3. 거리별 중앙값을 구해 CSV로 저장: `distance_m,rssi_dbm`
4. 회귀 실행:
   ```python
   from ble_positioning.distance.path_loss import fit_log_distance
   n, a = fit_log_distance(distances, rssis)   # 기울기=-n, 절편=A
   ```
5. **비콘 개체별로 반복** — P3 실측에서 같은 모델도 A가 -54~-58dBm으로 제각각.
   개체별 값은 `per_beacon_path_loss`에 넣는다.
6. 검증: 잔차 표준편차가 3dB 이상이면 환경(반사체) 재점검. n이 1.5~3.5 범위를
   벗어나면 측정 오류 의심 (P3 §III-C.1 typical 범위).

## 3. 인체 차폐 감쇠량 (Δ_shadow) 실측 — # ref: P1 §2.2, P3 §III-C.2

1. 비콘-폰 거리 2m, 4m 고정. LoS 100샘플 → 중앙값 = 기준선.
2. 사람 1명이 비콘-폰 직선상(폰에서 0.5m)에 서서 100샘플 → 중앙값.
3. Δ_shadow = 기준선 − 차폐값. 거리·인원수(1명/2~3명)별로 기록
   (P1 §2.2: 감쇠는 인체가 수신기에 가까울수록 큼).
4. `shadow_atten_db`에 대표값(2m 조건 권장), `shadow_drop_db`는 그 2/3 수준.

## 4. ANN 보정기 학습 데이터 (선택 — 파이프라인 승격용)

P1 §3 case 2 재현: 폐루프 경로를 2회 순회 — 1회는 차폐 없이, 1회는 동행자가 1m 앞에서
같은 경로로 걷게 하여 차폐 포함 데이터 수집. 70/30 분할로
`MlShadowCorrector.fit()` 학습 (tests/test_correction.py 의 합성 학습 코드 참조).

## 5. config 주입

```json
// config/site_gangnam_station.json (예시)
{
  "path_loss": {"n": 2.31, "a_dbm": -57.2},
  "shadow": {"shadow_atten_db": 8.4, "shadow_drop_db": 5.5},
  "filter": {"kalman_r": 5.1},
  "per_beacon_path_loss": {
    "B03": {"n": 2.45, "a_dbm": -55.9}
  }
}
```

```python
from ble_positioning.config.settings import load_config
cfg = load_config("config/site_gangnam_station.json")
```

주의: Eddystone 비콘의 광고 txPower는 **0m 기준** → 1m 환산(-41dB 관례)도
실측으로 확정할 것 (수신부 README 참조).
